## 先说它的核心逻辑

锂电池这个东西有个特点：长期满电或者长期亏电，都会加速老化。理想状态是保持在 30%～80% 之间循环。对于天天插电在桌子上当"台式机"用的 MacBook 用户来说，电池其实一直顶着 100% 的压力挂着，这才是真正的伤害源。

AlDente Pro 的做法是直接读写 MacBook 内置的 SMC 芯片，在系统层面控制充电行为——到了设定上限，它就让电脑直接走外接电源，不再充电池。这个机制比苹果自带的方案更及时、更可控。

## 核心功能拆解

**充电上限设置**：这是最基础也最重要的功能，设个 80%，以后插电也不会充过这个数。免费版就有，Pro 版在这基础上加了一堆料。

**过热保护**：温度高的时候（比如渲染、跑代码、高负载任务）自动暂停充电。锂电池在高温下充电是伤害最大的场景之一，这个功能在 Intel MacBook 上尤其实用。

**航行模式**：设置一个充电区间（比如 60%～80%），让电量始终在这个范围内小幅波动，不频繁大充大放，更接近教科书式的电池养护方案。

**多模式放电**：即使插着电，也能让电脑主动消耗电池电量。手动、自动、合盖放电都有，合盖状态下也能继续跑放电。

**电池校准**：长期保持低电量、不做完整充放循环，有时候系统会"误判"电量，50% 就突然关机。校准模式会自动跑一次完整循环来修正这个问题。

**一键 Top Up**：出门前点一下，立刻充满到 100%，确保外出续航。充满后可以自动恢复到正常的充电上限设置。

**MagSafe 灯控制**：可以控制 MagSafe 指示灯的充电状态反馈，细节控爱了。

**快捷指令支持**：和苹果快捷指令打通，可以做各种自动化组合——比如"外出工作"场景自动切换充电模式，"回家"场景自动恢复上限。
